# Offerto PROFI — Landing Page

Official landing page for **Offerto PROFI**, the iOS app that aggregates weekly deals from Switzerland's leading Cash & Carry wholesalers.

🌐 **Live:** [offerto-profi.ch](https://offerto-profi.ch)

---

## About

Offerto PROFI aggregates weekly offers from **Aligro**, **TopCC** and **Transgourmet** into a single fast app. Hundreds of deals, always up to date — built for gastronomy professionals.

**Key features:**
- All weekly offers from 3 wholesalers in one place
- Search & filter by category, store or discount
- Personal shopping list with offline support
- Digital product catalogs (Prospekte)
- Push notifications for new offers
- Available in DE / FR / IT / EN

---

## Tech Stack

- [Next.js 16](https://nextjs.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)

- [Vercel Analytics](https://vercel.com/analytics)
